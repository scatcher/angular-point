deleteAttachment
service in module ListItem

Description

Delete an attachment from a list item.

Usage

deleteAttachment(url);

Parameters

ParamTypeDetails
urlstring

Requires the URL for the attachment we want to delete.

Returns

object

Promise which resolves with the updated attachment collection.

Example

$scope.deleteAttachment = function (attachment) {
    let confirmation = window.confirm("Are you sure you want to delete this file?");
    if (confirmation) {
        scope.listItem.deleteAttachment(attachment).then(function () {
            alert("Attachment successfully deleted");
        });
    }
};
