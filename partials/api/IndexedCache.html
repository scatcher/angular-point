IndexedCache

Description

Attempts to be very similar to the ES6 Map instance. Cache constructor that is extended to make it easier to work with via prototype methods. Located in apIndexedCacheFactory.

Dependencies

  • apIndexedCacheFactory

Usage

IndexedCache([object]);

Parameters

ParamTypeDetails
[object]object

Optionally extend new cache with provided object.
