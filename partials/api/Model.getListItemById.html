getListItemById
service in module Model

Description

Inherited from Model constructor Attempts to retrieve the requested list item from the server.

Usage

getListItemById(entityId, options);

Parameters

ParamTypeDetails
entityIdnumber

Id of the item being requested.

optionsobject

Used to override apDataService defaults.

Returns

object

Promise that resolves with the requested list item if found. Otherwise it returns null. When working offline it returns a mock entity and replaces the mock id with the id provided.

Example

//Taken from a fictitious projectsModel.js
projectModel.getListItemById().then(function(entity) {
    //Do something with the located entity
    $scope.project = entity;
};
