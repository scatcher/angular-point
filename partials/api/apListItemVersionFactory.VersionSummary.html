VersionSummary
service in module apListItemVersionFactory

Description

Used specifically to determine difference between 2 distinct versions of a list item using the version history. Extends FieldChangeSummary.

Usage

VersionSummary(newerVersion, previousVersion);

Parameters

ParamTypeDetails
newerVersionIListItemVersion<T>

Updated version of list item.

previousVersionIListItemVersion<T>

Previous version of list item.

(default: {})
