registerPreDeleteAction
service in module ListItem.prototype

Description

Register a function on the list item prototype that is executed prior to deleting. Good use case is to perform cleanup prior to deleting or determining if user can delete. Method returns boolean and if true delete will continue, otherwise delete is prevented. There is no ListItem.registerPostDeleteAction because the list item no longer exists.

Usage

registerPreDeleteAction(action);

Parameters

ParamTypeDetails
actionFunction

Function that accepts no arguments and returns a boolean determining if delete can continue.

Returns

Function

Function that can be called to unregister.

Example

//In example projectsModel.ts
 export class Project extends ap.ListItem<Project>{
     title: string;
     users: User[];
     ...some other expected attributes
     constructor(obj) {
         super(obj);
         _.assign(this, obj);
     }
 }

 let unregister = Project.prototype.registerPreDeleteAction(function() {
     //Do some validation here and return true if user can delete
     //otherwise return false to prevent delete action
 });

 //At some point in the future if no longer necessary
 unregister();
