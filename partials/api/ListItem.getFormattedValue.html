getFormattedValue
service in module ListItem

Description

Given the attribute name on a listItem, we can lookup the field type and from there return a formatted string representation of that value.

Usage

getFormattedValue(fieldName, [options]);

Parameters

ParamTypeDetails
fieldNamestring

Attribute name on the object that contains the value to stringify.

[options]object

Pass through to apFormattedFieldValueService.getFormattedFieldValue.

Returns

string

Formatted string representing the field value.
