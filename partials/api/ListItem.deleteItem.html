deleteItem
service in module ListItem

Description

Deletes record directly from the object and removes record from user cache.

Usage

deleteItem();

Returns

object

Promise which really only lets us know the request is complete.

Example

<ul>
   <li ng-repeat="task in tasks">
       {{task.title}} <a href ng-click="task.deleteItem()>delete</a>
   </li>
</ul>

List of tasks. When the delete link is clicked, the list item item is removed from the local cache and the view is updated to no longer show the task.
