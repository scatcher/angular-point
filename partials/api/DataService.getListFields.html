getListFields
service in module DataService

Description

Returns field definitions for a specified list.

Usage

getListFields(options);

Parameters

ParamTypeDetails
optionsobject

Configuration parameters.

options.listNamestring

GUID of the list.

[options.webURL]string

URL to the site containing the list if differnt from primary data site in apConfig.

Returns

Promise

Promise which resolves with an array of field definitions for the list.
