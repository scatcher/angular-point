attrToJson
service in module utilityService

Description

Converts a SharePoint string representation of a field into the correctly formatted JavaScript version based on object type.

Usage

attrToJson(value, objectType);

Parameters

ParamTypeDetails
valuestring

SharePoint string.

objectTypestring

The type based on field definition. Options:[ DateTime, Lookup, User, LookupMulti, UserMulti, Boolean, Integer, Float, Counter, MultiChoice, Currency, Number, Calc, JSON, Text [Default] ]

(default: 'Text')

Returns

*

The formatted JavaScript value based on field type.
