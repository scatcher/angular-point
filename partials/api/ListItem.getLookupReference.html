getLookupReference
service in module ListItem

Description

Allows us to retrieve the listItem being referenced in a given lookup field.

Usage

getLookupReference(fieldName, lookupId);

Parameters

ParamTypeDetails
fieldNamestring

Name of the lookup property on the list item that references a listItem.

lookupIdnumber

The listItem.lookupId of the lookup object. This allows us to also use this logic on a multi-select by iterating over each of the lookups.

(default: listItem.fieldName.lookupId)

Returns

object

The listItem the lookup is referencing or undefined if not in the cache.

Example

var project = {
   title: 'Project 1',
   location: {
       lookupId: 5,
       lookupValue: 'Some Building'
   }
};

//To get the location listItem
var listItem = project.getLookupReference('location');
