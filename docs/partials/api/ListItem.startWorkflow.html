startWorkflow
service in module ListItem

Description

Given a workflow name or templateId we initiate a given workflow using apDataService.startWorkflow.

Usage

startWorkflow(options);

Parameters

ParamTypeDetails
optionsobject

Params for method and pass through options to apDataService.startWorkflow.

[options.templateId]string

Used to directly start the workflow without looking up the templateId.

[options.workflowName]string

Use this value to lookup the templateId and then start the workflow.

Returns

promise

Resolves with server response.
