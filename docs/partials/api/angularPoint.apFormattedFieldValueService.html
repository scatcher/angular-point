apFormattedFieldValueService
service in module angularPoint

Description

Returns the formatted string value for a field based on field type.

Methods

  • getFormattedFieldValue(prop, propertyType, options)

    Parameters
    ParamTypeDetails
    propobjectarraystringintegerboolean

    Target that we'd like to stringify.

    propertyTypestring

    Assumes by default that it's already a string. Most of the normal field types identified in the model field definitions are supported.

    • Lookup
    • User
    • Boolean
    • DateTime
    • Integer
    • Number
    • Counter
    • MultiChoice
    • UserMulti
    • LookupMulti

    (default: 'String')

    optionsobject

    Optional config.

    options.delimstring

    Optional delimiter to split concatenated strings.

    (default: ', ')

    options.dateFormatstring

    Either 'json' which converts a date into ISO8601 date string or a mask for the angular date filter.

    (default: 'short')

    Returns
    string

    Stringified property on the object based on the field type.

    Example

    var project = {
  title: 'Super Project',
 members: [
   { lookupId: 12, lookupValue: 'Joe' },
   { lookupId: 19, lookupValue: 'Beth' }
 ]
};

var membersAsString = apFormattedFieldValueService:getFormattedFieldValue({
  project.members,
  'UserMulti',
  { delim: ' | '} //Custom Delimiter
});

// membersAsString = 'Joe | Beth';
