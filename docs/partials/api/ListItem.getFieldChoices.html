getFieldChoices
service in module ListItem

Description

Uses the field definition defined in the model to attempt to find the choices array for a given Lookup or MultiLookup type field. The default value is fieldDefinition.choices which can optionally be added to a given field definition. If this isn't found, we check fieldDefinition.Choices which is populated after a GetListItemsSinceToken operation or a Model.extendListMetadata operation. Finally if that isn't available we return an empty array.

Usage

getFieldChoices(fieldName);

Parameters

ParamTypeDetails
fieldNamestring

Internal field name.

Returns

string[]

An array of choices for a Choice or MultiChoice type field.
