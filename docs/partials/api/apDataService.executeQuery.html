executeQuery
service in module apDataService

Description

Primary method of retrieving list items from SharePoint. Look at Query and Model for specifics.

Usage

executeQuery(model, query, [options]);

Parameters

ParamTypeDetails
modelobject

Reference to the model where the Query resides.

queryobject

Reference to the Query making the call.

[options]object

Optional configuration parameters.

options.targetArray

The target destination for returned entities

(default: model.getCache())

options.offlineXMLstring

Optionally include the location of a custom offline XML file specifically for this query.

(default: apConfig.offlineXML + model.list.title + '.xml')

Returns

object
  • Key value hash containing all list item id's as keys with the entity as the value.
