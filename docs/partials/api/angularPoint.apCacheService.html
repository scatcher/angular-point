apCacheService
service in module angularPoint

Description

Stores a reference for all list items based on list GUID and list item id. Allows us to then register promises that resolve once a requested list item is registered in the future.

  • registerEntity(entity, [targetCache])

    Registers an entity in the cache and fulfills any pending deferred requests for the entity. If the entity already exists in the cache, we extend the existing object with the updated entity and return a reference to this updated object so the there is only a single instance of this entity withing the cache.

    Parameters
    ParamTypeDetails
    entityobject

    Pass in a newly created entity to add to the cache.

    [targetCache]object

    Optionally pass in a secondary cache to add a reference to this entity.
