User

Description

Allows for easier distinction when debugging if object type is shown as a User. Turns a delimited ";#" string into an object shown below depeinding on field settings: 

{
     lookupId: 1,
     lookupValue: 'Joe User'
}
or 
{
     lookupId: 1,
     lookupValue: 'Joe User',
     loginName: 'joe.user',
     email: 'joe@company.com',
     sipAddress: 'whatever',
     title: 'Sr. Widget Maker'
}

Usage

User(s);

Parameters

ParamTypeDetails
sstring

Delimited string used to create a User object.
