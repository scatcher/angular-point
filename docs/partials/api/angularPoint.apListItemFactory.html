apListItemFactory
service in module angularPoint

Description

Exposes the ListItem prototype and a constructor to instantiate a new ListItem. See ListItem for details of the methods available on the prototype.

Dependencies

  • ListItem
  • apCacheService
  • apDataService
  • apUtilityService

Methods

  • create()

    Instantiates and returns a new ListItem.

  • createGenericFactory()

    In the event that a factory isn't specified, just use a standard constructor to allow it to inherit from ListItem
