setPristine
service in module ListItem

Description

Resets all list item properties back to a pristine state but doesn't update any properties added manually to the list item.

Usage

setPristine([listItem]);

Parameters

ParamTypeDetails
[listItem]ListItem

Optionally pass initial list item object back to the list item constructor to run any initialization logic. Otherwise we just overwrite existing values on the object with a copy from the original object.
